![Screenshot of the project](https://i.ibb.co/g6FhZp5/image.png)

### Installation

```git clone https://github.com/miojoepao/financemgmt.git```

```cd financemgmt```

```yarn add package.json```

```yarn dev```

concatenated for quickness sake
